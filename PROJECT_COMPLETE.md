### Technology Stack

```
Frontend: Next.js 16 + React 19 + shadcn/ui + Tailwind

Runtime: TypeScript + Bun (blazing fast)
Security: WASM (Extism) + Containers (gVisor optional)

Testing: Vitest + Playwright
Monitoring: Sentry + Prometheus
Deployment: Vercel (frontend) + Convex (backend)
```

### Key Innovations

1. **Complexity-Based Model Selection**
   - Analyzes each message (0-100 score)
   - Routes to cheapest capable model
   - Saves ~60% on AI costs

2. **Agent Dreams**
   - Background processing while you sleep
   - Consolidates memories
   - Extracts entities
   - Updates knowledge graphs
   - Wake up to smarter agent

3. **Multi-AI Consensus**
   - Ask 3 models in parallel
   - Vote on best answer
   - Return consensus response
   - Reduces hallucinations

4. **Time Travel Debug**
   - Replay conversations
   - What-if scenarios
   - Debug agent reasoning
   - Improve prompts
