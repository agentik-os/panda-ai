# Agentik OS Swag Design Specifications

This document provides design specifications for all Agentik OS swag items. Use these specs when creating or ordering swag.

---

## Brand Guidelines

### Logo Usage
- **Primary Logo**: Full color on white/light backgrounds
- **Inverse Logo**: White on dark backgrounds
- **Monochrome**: Black or white only when color not available
- **Minimum Size**: 20px height for digital, 0.5" for print

### Colors

| Name | Hex | RGB | CMYK | Pantone | Usage |
|------|-----|-----|------|---------|-------|
| Primary Blue | `#0066FF` | 0, 102, 255 | 100, 60, 0, 0 | 2727 C | Main brand color |
| Dark Blue | `#0047B3` | 0, 71, 179 | 100, 80, 0, 0 | 2756 C | Text, dark mode |
| Light Blue | `#3385FF` | 51, 133, 255 | 80, 48, 0, 0 | 2718 C | Accents |
| Gray | `#333333` | 51, 51, 51 | 0, 0, 0, 80 | Black 7 C | Body text |
| Light Gray | `#F5F5F5` | 245, 245, 245 | 0, 0, 0, 4 | Cool Gray 1 C | Backgrounds |

### Typography
- **Headings**: Inter Bold
- **Body**: Inter Regular
- **Code**: JetBrains Mono

## Production Guidelines

### Print Files
- **Format**: Vector (AI, EPS, SVG)
- **Resolution**: Raster at least 300 DPI
- **Color Mode**: CMYK for print, RGB for digital
- **Bleed**: 0.125" on all sides
- **Safe Area**: 0.25" from trim

### Minimum Orders
- T-shirts: 50 units per design
- Hoodies: 25 units per design
- Stickers: 250 per design
- Pins: 100 per design
- Challenge coins: 50 units

### Quality Standards
- Color accuracy: ±5% Delta E
- Print registration: ±1mm
- Embroidery: 75% stitch density minimum
- Vinyl: 3-5 year outdoor durability

### Approved Vendors
- **T-shirts/Hoodies**: Printful, Bella+Canvas
- **Stickers**: Sticker Mule, StickerApp
- **Pins**: Wizardpins, PinMart
- **Challenge Coins**: CoinForce, PinMart
