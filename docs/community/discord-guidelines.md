## Channel Usage Guidelines

### #help-* Channels
When asking for help:
1. **Search first** - Check if your question was answered before
2. **Provide context** - OS, version, error messages, reproduction steps
3. **Use code blocks** - Format code with \`\`\`language
4. **Share screenshots** - Visual issues need visual proof
5. **Mark as solved** - React with ‚úÖ when your issue is resolved

### #showcase-* Channels
When showcasing:
1. **Provide description** - What does it do?
2. **Share code/links** - GitHub repo, demo video, screenshots
3. **Explain unique features** - What makes it special?
4. **Credit dependencies** - Acknowledge libraries/skills used

### #bug-reports Channel
Bug report format:
```
**Bug:** Brief description
**Version:** Agentik OS version
**Environment:** OS, Node version
**Steps to reproduce:**
1. Step 1
2. Step 2
3. Step 3

**Expected behavior:** What should happen
**Actual behavior:** What actually happens
**Error messages:** Full error output
**Screenshots:** If applicable
```

### #feature-requests Channel
Feature request format:
```
**Feature:** Brief title
**Problem:** What problem does this solve?
**Proposed solution:** How should it work?
**Alternatives considered:** Other approaches you thought of
**Priority:** Low/Medium/High (from your perspective)
**Willing to contribute:** Yes/No
```

## Community Expectations

### Be Respectful
- Treat everyone with respect and kindness
- No harassment, hate speech, or discrimination
- Disagree constructively, never personally
- Welcome newcomers and help them learn

### Be Helpful
- Share knowledge freely
- Answer questions when you can
- Point people to resources
- Contribute back to the community

### Be Professional
- Keep discussions on-topic
- No spam or self-promotion (unless in appropriate channels)
- No NSFW content
- Respect intellectual property

### Be Collaborative
- Review PRs constructively
- Test features and provide feedback
- Share your learnings
- Participate in community events

## Moderation

### Warning System
1. **First offense**: Friendly warning
2. **Second offense**: Temporary mute (24h)
3. **Third offense**: Temporary ban (7 days)
4. **Severe/Repeated offenses**: Permanent ban

### Report Issues
- Use Discord's report feature
- DM any moderator
- Email: community@agentik-os.com

### Appeal Process
If you believe a moderation action was unfair:
1. Wait 24 hours to cool down
2. DM a different moderator
3. Explain your perspective calmly
4. Provide context if needed
