## Call Format

### Typical Agenda (60 minutes)

**5:00 - 5:05 PM** (5 min) - Welcome & Intros
- Quick intro from host
- New community members welcome
- Overview of agenda

**5:05 - 5:20 PM** (15 min) - Project Updates
- Development progress
- New features shipped
- Upcoming roadmap items
- Key metrics (stars, contributors, downloads)

**5:20 - 5:45 PM** (25 min) - Community Q&A
- Questions from Discord chat
- Technical questions
- Feature requests discussion
- General feedback

**5:45 - 5:55 PM** (10 min) - Community Showcase
- 1-2 community members demo their work
- Cool use cases
- Interesting integrations

**5:55 - 6:00 PM** (5 min) - Wrap-up
- Recap action items
- Announce next call
- Thank everyone

## How to Join

### Discord (Live)
1. Join our [Discord server](https://discord.gg/agentik-os)
2. Navigate to #voice-community-call channel
3. Join voice on call day at 5:00 PM UTC

### YouTube (Live Stream)
1. Subscribe to [Agentik OS YouTube](https://youtube.com/@AgentikOS)
2. Live stream link posted in Discord 30 min before call
3. Watch live or catch recording later

### Can't Attend Live?
- Recordings posted to YouTube within 24 hours
- Summary and action items in Discord #announcements
- Q&A transcript published to GitHub Discussions

## Community Showcase Guidelines

Want to demo your work on a call?

**Requirements:**
- 5-10 minute demo
- Built with/for Agentik OS
- Prepared demo (not live coding unless requested)
- Tested beforehand

**How to apply:**
1. Post in #community-calls channel
2. Include:
   - What you built
   - Why it's interesting
   - Demo outline
   - Link to repo/live demo

**Selection:** First-come-first-served, with preference for diverse use cases

**Deadline:** 1 week before call
