# AOCM - Agentik OS Certified Marketer

## Overview

The AOCM certification validates proficiency in leveraging AI agents for marketing, sales, and business operations on the Agentik OS platform. This 10-hour program covers use cases, agent design, prompt engineering, and measuring ROI â€” no coding required.

## Prerequisites

- Basic understanding of marketing concepts (SEO, email, social media)
- Familiarity with AI chatbots (ChatGPT, Claude, etc.)
- No programming experience required

## Curriculum Structure

| Module | Title | Duration | Focus |
|--------|-------|----------|-------|

## Module Details

### Module 1: AI Agent Use Cases (3 hours)

**Topics:**
- AI agents in marketing: overview and landscape
- Customer support automation (FAQ, ticket routing, escalation)
- Lead generation and qualification agents
- Content creation and curation workflows
- Social media management and scheduling
- Email marketing personalization
- SEO analysis and optimization agents
- Sales enablement: proposal generation, follow-ups
- Multi-channel strategy: CLI, API, Telegram, Discord, Web

**Hands-On Labs:**
- Lab 1.1: Explore the Agentik OS dashboard
- Lab 1.2: Chat with pre-built marketing agents
- Lab 1.3: Map your marketing workflow to agent capabilities

**Assessment:** Quiz (15 questions)

### Module 2: Agent Design for Non-Developers (3 hours)

**Topics:**
- Agent anatomy: system prompt, skills, model selection
- Choosing the right AI model for your use case
- Prompt engineering fundamentals for marketers
- System prompt best practices (tone, persona, constraints)
- Installing marketplace skills (no code required)
- Configuring agent behavior through the dashboard
- Multi-agent workflows: chaining agents for complex tasks
- Brand voice consistency across agents
- Guardrails: preventing off-brand or harmful outputs

**Hands-On Labs:**
- Lab 2.1: Create a customer support agent via dashboard
- Lab 2.2: Write effective system prompts for marketing
- Lab 2.3: Install and configure marketplace skills
- Lab 2.4: Set up a multi-agent content pipeline

**Assessment:** Agent configuration submission (graded)

### Module 3: Measuring Success (2 hours)

**Topics:**
- Key metrics for AI agent performance
- Cost tracking and budget management
- Conversation quality scoring
- Customer satisfaction measurement (CSAT, NPS)
- A/B testing agent configurations
- ROI calculation for agent deployment
- Dashboard analytics and reporting
- Optimization strategies based on data

**Hands-On Labs:**
- Lab 3.1: Set up cost alerts and budgets
- Lab 3.2: Configure analytics dashboard
- Lab 3.3: Run an A/B test between two agent configurations

**Assessment:** Analytics report submission

### Module 4: Hands-On Capstone (2 hours)

**Project: Deploy a Marketing Agent Stack**

Requirements:
1. Create a customer-facing agent with appropriate persona
2. Configure at least 2 marketplace skills
3. Set up cost tracking and budget alerts
4. Write a system prompt that maintains brand voice
5. Test the agent across at least 2 channels
6. Document expected ROI and success metrics

**Deliverables:**
- Working agent accessible via dashboard
- System prompt document
- Channel configuration
- ROI projection document
- 5-minute presentation (written or video)

**Assessment:** Capstone review (manual grading)
