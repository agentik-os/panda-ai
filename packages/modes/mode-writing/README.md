# Writing Mode ✍️

**Content creation with SEO optimization, storytelling, and professional editing.**

## Features

- **Blog Posts:** SEO-optimized long-form content
- **Social Media:** Engaging posts for Twitter, LinkedIn, Instagram
- **Copywriting:** Persuasive landing pages and sales copy
- **Editing:** Professional proofreading and refinement

## Agents

### 1. Content Writer
Creates engaging blog posts and articles using AIDA formula.

### 2. SEO Specialist
Optimizes content for search engines (keyword research, on-page SEO).

### 3. Editor
Refines content for clarity, grammar, and impact.

## Recommended Skills

- `web-search` - Research topics and trends
- `keyword-research` - Find SEO keywords
- `grammar-check` - Check grammar and style
- `file-operations` - Save drafts

## Example Workflows

### Blog Post (2-4 hours)
```
1. Keyword research
2. Analyze SERP top 10
3. Create outline
4. Write hook and body
5. SEO optimization
6. Edit and proofread
```
