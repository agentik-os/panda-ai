# Building Your First AI Agent in 5 Minutes

**What you'll build:** A web research agent that can search the internet, summarize findings, and track costs in real-time.

**Prerequisites:** Node.js 18+ (or Docker)

**Time:** 5 minutes

## Step 1: Install Agentik OS (60 seconds)

### Option A: Docker (Fastest)

```bash
docker run -p 3000:3000 agentikos/agentik-os
```

Open `http://localhost:3000` ‚Üí Dashboard opens ‚Üí **Done!**

### Option B: npm/pnpm

```bash
npm install -g @agentik-os/cli
# or
pnpm install -g @agentik-os/cli
```

Verify installation:

```bash
panda --version
# Output: @agentik-os/cli v1.0.0
```

## Step 2: Initialize Project (30 seconds)

Create a new agent project:

```bash
panda init web-researcher
cd web-researcher
```

## Step 4: Create Your First Agent (30 seconds)

```bash
panda agent create web-researcher \
  --model claude-sonnet-4-5 \
  --description "AI agent that researches topics on the web"
```

This creates `agent.yaml`:

```yaml
name: web-researcher
description: AI agent that researches topics on the web

model:
  primary: claude-sonnet-4-5
  fallback: gpt-4o

routing:
  enabled: true
  strategy: cost-optimized

budget:
  daily_limit: 10.00    # $10/day max
  alert_threshold: 0.80  # Alert at 80%

skills:
  - web-search
  - summarize

system_prompt: |
  You are a web research assistant. When given a topic:
  1. Search the web for relevant, up-to-date information
  2. Analyze and synthesize findings
  3. Provide a clear, structured summary
  4. Cite all sources
```

## Advanced: Multi-Model Routing

Save costs by routing simple queries to cheap models:

Edit `agent.yaml`:

```yaml
routing:
  enabled: true
  strategy: cost-optimized
  models:
    trivial: gpt-4o-mini      # $0.001/query
    simple: gpt-4o            # $0.01/query
    complex: claude-sonnet    # $0.03/query
    critical: claude-opus     # $0.15/query

  classification:
    threshold:
      trivial: 0.3   # "What time is it?" ‚Üí mini
      simple: 0.6    # "How do I...?" ‚Üí 4o
      complex: 0.8   # "Analyze this..." ‚Üí sonnet
```

Restart agent:

```bash
panda agent restart web-researcher
```

Now watch costs drop by 50-80%!

## Advanced: Custom Skills

Create a custom skill:

```bash
panda skill create summarize
```

Edit `skills/summarize/index.ts`:

```typescript
import { Skill } from '@agentik-os/sdk'

export const summarize: Skill = {
  name: 'summarize',
  description: 'Summarize long text into key points',

  parameters: {
    text: {
      type: 'string',
      description: 'Text to summarize',
      required: true,
    },
    max_points: {
      type: 'number',
      description: 'Maximum bullet points',
      default: 5,
    },
  },

  async execute({ text, max_points }) {
    // Your summarization logic here
    const points = await extractKeyPoints(text, max_points)
    return {
      summary: points.join('\n'),
      word_count: text.split(' ').length,
    }
  },
}
```

Install your custom skill:

```bash
panda skill install ./skills/summarize
```

## Advanced: Agent Dreams (Autonomous Mode)

Let your agent work while you sleep:

```bash
panda agent create night-researcher \
  --mode creative \
  --schedule "0 2 * * *" \
  --autonomous true
```

Configure the task in `agent.yaml`:

```yaml
autonomous:
  enabled: true
  schedule: "0 2 * * *"  # 2 AM daily

  tasks:
    - name: "Daily Tech News Summary"
      prompt: |
        Research the top 10 tech news stories from the last 24 hours.
```

## Production Checklist

Before deploying to production:

- [ ] Set environment variables (API keys, secrets)
- [ ] Configure budget limits (`daily_limit`, `monthly_limit`)
- [ ] Enable monitoring (`panda monitoring enable`)
- [ ] Set up alerts (`panda alerts create --type budget`)
- [ ] Review security settings (`panda security audit`)
- [ ] Test error handling (`panda test --scenarios error`)

## Troubleshooting

### "API key not found"

Make sure `.env` exists with valid keys:

```bash
cat .env
# Should show: OPENAI_API_KEY=sk-...
```

### "Skill not found: web-search"

Reinstall the skill:

```bash
panda skill install web-search --force
```

### "Budget limit exceeded"

Reset or increase budget:

```bash
panda agent update web-researcher --daily-limit 20.00
```

### "Model not available"

Check model availability:

```bash
panda models list --available
```
