# Introducing Agentik OS: The AI Agent Operating System

**TL;DR:** OpenClaw got 191K GitHub stars but wasn't production-ready. We built Agentik OS to be what developers actually need: a robust, scalable, enterprise-grade AI Agent Operating System.

## The Problem with Current AI Agent Platforms

We've been building AI agents for 2 years. We've tried them all:

- **Langchain**: Great for prototypes, terrible for production
- **OpenClaw**: Amazing vision, but 341 malicious skills proved security was an afterthought
- **n8n/Zapier**: Not built for AI agents, just workflow automation
- **Custom solutions**: Reinventing the wheel every project

Every platform had the same problems:

### 1. No Cost Control

Running AI agents 24/7 with Claude Opus? **$12K/month** for a single agent. No visibility into costs until the bill arrives.

### 2. Security Nightmares

The ClawHavoc incident (341 malicious skills) showed what happens when you don't sandbox skills properly. One rogue skill can exfiltrate your entire database.

### 3. Model Lock-In

Married to OpenAI? Hope they don't change pricing. Want to use Gemini 2.5? Rewrite everything.

### 4. Zero Observability

"Why did my agent cost $500 today?" Good luck finding out.

## Enter Agentik OS

We built Agentik OS to solve these problems. It's an **operating system** for AI agents, not just another framework.

### What Makes It an OS?

1. **Process Management**: Agents run as isolated processes
2. **Resource Control**: CPU, memory, and **token limits** per agent
3. **Scheduling**: OS Modes (Focus/Creative/Research) optimize for speed vs quality
4. **Security**: WASM + gVisor sandbox for untrusted skills
5. **Observability**: Real-time cost tracking, logs, metrics

Think of it as **Linux for AI agents**.

## Key Features

### 1. Multi-Model Router

Save **72% on AI costs** by routing queries to the right model:

```typescript
// Simple query â†’ GPT-4o-mini ($0.001)
agent.chat("What's 2+2?")  // Routed to mini

// Complex query â†’ Claude Opus ($0.15)
agent.chat("Analyze this 50-page contract")  // Routed to Opus

// Image query â†’ Gemini 2.5 Pro
agent.chat("What's in this image?")  // Routed to Gemini
```

**Result:** $12K/month â†’ $3.4K/month for TechCorp (case study coming).

Know **exactly** what you're spending:

Set budget alerts: `panda agent create --budget-limit 50`

### 3. Skill Marketplace

200+ pre-built skills, all **security-scanned**:

- Web search (Google, Bing, DuckDuckGo)
- File operations (read, write, search)
- Database queries (PostgreSQL, MySQL, MongoDB)
- API calls (REST, GraphQL, gRPC)
- Email (Gmail, SendGrid, SMTP)
- Calendar (Google Calendar, Outlook)

Install with one command:

```bash
panda skill install web-search
panda skill install google-calendar
```

All skills run in **WASM sandbox**. No system access unless explicitly granted.

### 4. Agent Dreams

Your agents work while you sleep:

```bash
panda agent create night-worker \
  --mode creative \
  --schedule "0 2 * * *" \  # 2 AM daily
  --task "Generate blog post ideas"
```

Wake up to:
- Blog post drafts
- Competitor analysis
- Market research
- Bug triage reports

### 5. Time Travel Debug

Rewind agent execution to any point:

```bash
panda debug time-travel \
  --agent customer-support \
  --timestamp "2027-01-15T14:23:00Z"
```

See:
- Full conversation history
- Model responses
- Skill executions
- Cost breakdown
- Token usage

Debug production issues **without guessing**.

## Getting Started

### Docker (Fastest)

```bash
docker run -p 3000:3000 agentikos/agentik-os
```

Open `http://localhost:3000` â†’ done.

### npm/pnpm

```bash
npm install -g @agentik-os/cli
panda init my-agent
cd my-agent
panda dev
```

### From Source

```bash
git clone https://github.com/agentik-os/agentik-os.git
cd agentik-os
pnpm install
pnpm dev
```
