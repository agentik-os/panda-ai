## Product Description

### Elevator Pitch (30 seconds)

Agentik OS is an open-source operating system for AI agents. It's like Linux for autonomous AI systems - providing multi-model routing, real-time cost tracking, sandboxed skill execution, and production-grade infrastructure. Developers can build agents in minutes, not months, and deploy them confidently to production.

### Full Description (2 minutes)

Agentik OS is the first production-ready operating system designed specifically for AI agents. It solves the three biggest problems developers face when building AI systems: vendor lock-in, cost unpredictability, and production reliability.

**Multi-Model Router:** Intelligently routes requests to the optimal AI model (Claude, GPT, Gemini, Ollama) based on query complexity, saving developers 50-80% on AI costs. A simple "What's 2+2?" goes to GPT-4o-mini ($0.001), while "Analyze this 50-page contract" routes to Claude Opus ($0.15).

**Real-Time Cost Tracking:** See exactly what you're spending as you build. Built-in dashboard shows cost per query, model usage breakdown, and budget alerts. No more surprise $10K bills.

**Secure Skill Marketplace:** 200+ pre-built skills (web search, file operations, calendar, email, payments) that run in WASM sandboxes with gVisor isolation. One line of code to install, zero system access unless explicitly granted.

**Agent Dreams:** Your agents work while you sleep. Schedule autonomous tasks (market research, competitor analysis, report generation) to run nightly and wake up to completed work.

**Production Features:** Event-sourced architecture, time-travel debugging, multi-tenancy, Kubernetes deployment, OAuth2 authentication, and comprehensive monitoring.

Built with modern stack: Bun runtime, Next.js 16 dashboard, Convex backend, TypeScript throughout. 100% open source (MIT license).

## Media FAQ

### General

**Q: What is Agentik OS?**
A: Agentik OS is the first production-ready operating system designed specifically for AI agents. It provides multi-model routing, cost tracking, secure skill execution, and enterprise-grade infrastructure - all open source.

**Q: Who is Agentik OS for?**
A: Developers building AI agents - from indie hackers to enterprise teams. If you're using Claude, GPT, or Gemini APIs to build autonomous systems, Agentik OS saves you months of infrastructure work.

**Q: How is it different from Langchain?**
A: Langchain is great for prototyping, but Agentik OS is built for production. We provide cost tracking, security sandboxing, multi-model routing, and enterprise features out of the box.

**Q: Why did you build this?**
A: We saw developers at Anthropic and OpenAI struggle with the same problems: cost explosions, vendor lock-in, and production reliability. We built the infrastructure we wished existed.

### Technical

**Q: What languages/frameworks does it support?**
A: Agentik OS is written in TypeScript/Bun but agents can be built in any language. The skill SDK supports TypeScript, Python, and Go. WebAssembly skills can be written in Rust, C++, or any WASM-compatible language.

**Q: Can I use my own AI models?**
A: Yes! Supports Claude, GPT-4/5, Gemini, and Ollama (for local models). The router automatically selects the best model based on your configuration.

**Q: How does cost tracking work?**
A: We intercept all AI API calls and calculate costs in real-time based on input/output tokens. Dashboard shows per-query costs, model breakdowns, and trends. Budget alerts prevent overspending.

**Q: Is it really secure after ClawHavoc?**
A: Yes. Every skill runs in a WASM sandbox with gVisor isolation. Skills have zero system access unless explicitly granted through the permission system. We learned from ClawHavoc's 341 malicious skills.

**Q: Can I deploy on-premises?**
A: Yes! Agentik OS is fully self-hostable. We provide Docker images, Kubernetes manifests, and Helm charts. No data leaves your infrastructure if you choose to self-host.

**Q: What's the architecture?**
A: Event-sourced architecture with Convex backend, Next.js 16 dashboard, Bun runtime. Skills run in WASM (Extism) with gVisor. Supports horizontal scaling, multi-tenancy, and high availability.

### Business

**Q: How do you make money if it's open source?**
A: Managed hosting ($49+/month), enterprise support, and eventually enterprise features (SSO, advanced compliance, etc.) while keeping core platform free forever.

**Q: Will you stay open source?**
A: Yes, 100%. Core platform is MIT licensed and will always be free. We believe the best infrastructure should be open and community-driven.

**Q: Are you hiring?**
A: Yes! See careers page at agentik-os.com/careers. We're remote-first and looking for backend engineers, DevRel, and community managers.

**Q: How can companies get involved?**
A: Join our Discord, contribute to GitHub, or sponsor development through GitHub Sponsors. Enterprise customers can reach out to enterprise@agentik-os.com for support contracts.

### Community

**Q: How can I contribute?**
A: GitHub (code), Discord (community support), documentation writing, skill development, or blog posts. See CONTRIBUTING.md on GitHub.

**Q: Is there a roadmap?**
A: Yes! Public roadmap at agentik-os.com/roadmap. Community can vote on features. Q1 2027 focus: Knowledge Graph (Neo4j), Voice Agents, Visual Workflows.

**Q: How many contributors do you have?**
A: 50+ open source contributors and growing. We recognize top contributors monthly and offer swag, credits, and co-marketing opportunities.
